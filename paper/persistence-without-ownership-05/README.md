# Persistence Without Ownership — V5 external-review package

This directory is a no-new-experiment synthesis built from protected committed evidence. The primary conclusion is:

> An engineered experience-dependent state remains causally active through deep material turnover but fails the preregistered local-ownership criterion. Within the tested C1c architecture, causal persistence is insufficient for individual memory.

## Primary files

- `PERSISTENCE_WITHOUT_OWNERSHIP_05.tex` — 7,000–9,000-word journal-shaped manuscript.
- `PERSISTENCE_WITHOUT_OWNERSHIP_05_SUPPLEMENT.tex` — standalone supplementary information.
- `CLAIM_LEDGER_05.md` — exact claim classes and boundaries.
- `NUMERICAL_RECONCILIATION_05.md/.csv` — headline values and exact checks.
- `EXPERIMENT_PROVENANCE_MAP_05.md` — protected lineage and evidential roles.
- `FIGURE_INDEX_05.md` and `figures/` — eight source-bound figures.
- `REPRODUCIBILITY_05.md` — no-engine rebuild and PDF-QA instructions.
- `RAW_DATA_REGISTRY_05.json` and `SOURCE_BINDINGS_05.json` — exact source registry.
- `REFERENCE_VERIFICATION_05.json` and `references.bib` — 30 DOI-verified references.
- `COVER_LETTER_DRAFT_05.md` and `RESUME_GRAND_PUBLIC_FR_05.md` — communication drafts.
- `HOSTILE_PEER_REVIEWS_05.md` and `RESPONSE_TO_REVIEWERS_DRAFT_05.md` — adversarial review package.
- `PACKAGE_VALIDATION_05.json` and `PDF_VISUAL_QA_REPORT_05.md` — package and rendered-PDF checks.

Compiled PDFs live in repository-level `output/pdf/`.

## Version disposition
