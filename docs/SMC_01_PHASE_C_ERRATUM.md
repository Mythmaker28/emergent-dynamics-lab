# SMC-01 — Phase C Interpretive Erratum (committed before SMC-01 runs)

Three wording corrections to the Phase C record. Observed numbers stand; scope of claims narrowed.

## Qualification A — rank gate
Preregistered rank gate was σ₂/σ₁ ≥ 0.30. Observed prospective σ₂/σ₁ = 0.283.
Correct wording: **STRONG EVIDENCE FOR TWO DECODED HISTORY COORDINATES; ORIGINAL RANK GATE: STRICT FAIL, MARGINALLY.**
The 0.30 threshold is NOT retroactively changed. SMC-01 re-tests storage rank against a matched null on a fresh family.

## Qualification B — turnover
Material turnover was not re-measured under C1c; C1c changed the memory dynamics (eta_w, eta_d1, eta_d2, k_exp),
so the MCM M≈0.12 cannot be inherited. Wording: **C1c TURNOVER SURVIVAL: NOT ESTABLISHED** (SMC-01 measures it).

## Qualification C — mean transplant
Mean-only transplant destroys spatial information by construction, so it cannot show a spatial memory is
non-transplantable. Replace "h2 is not transplantable" with: **h2 is not preserved by mean-only transplantation;
structured transplantability remains untested** (SMC-01 tests full-field transplant).

## Added caution (this agent) — is h2 even "spatial"?
Phase C decoded h2 at R²≈0.62 from the entity-MEAN (m1,m2) pair and ≈0.90 from distributional statistics
(mean/std/percentiles over entity cells). Those statistics are permutation-invariant (frame-free) but NOT
necessarily geometric. SMC-01 must therefore test explicitly whether h2 lives in (a) the 2-scalar mean pair,
(b) frame-free value-dispersion, or (c) a genuine entity-relative geometric pattern — before any "spatial
organizational memory" claim.
