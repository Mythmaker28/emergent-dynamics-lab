## 1. Question and pre-committed claim boundary

Can two frozen histories bring exact clones of one organism to the same declared macro-observable
`Z = core mass`, yet — released under a common dynamics, boundary, `up_ref` and probe-free relaxation —
produce a different subsequent excursion of `Z`?

A supported result establishes **only**: (i) insufficiency of the selected macrostate `Z`; and
(ii) functionally relevant hidden history-dependent degrees of freedom. It does **not** establish
identical complete physical states, local ownership, individual identity, active reconstruction,
heredity, or agency. These exclusions are part of the preregistration, not caveats added later.

## 2. Primary observable, estimand, and unit (frozen)

- **`Z`** = core-region mass = `sum(rho over the frozen radius-10 core mask centred at the checkpoint
  focal centroid)`; tracker-free endpoint; identical support for both clones; **contains** (does not
  equal) the focal body mass (`core ≥ body`; DEV core ≈ 1.10–2.03× body, median 1.38×). Tracked body
  mass is a robustness co-observable only.
- **Per-branch hump** `K_b = Σ_s (Z_b(t_R^b+s) − Z_ref(t_ref+s))·Δs` over horizon `T_hor`.
- **Primary estimand** `D_w = K_OVER − K_APPR` per world (between-history hump difference; zero under
  `Z`-sufficiency; cancels the common reference).
- **Unit** = original source world. Branches/crossings/clones/time points never increase `n`.
- **Primary gate (unoriented, frozen):** `{D_w}` has ≥ 75 % common sign AND its 95 % Student-t interval
  excludes zero, across ≥ `N_min` complete worlds. Secondary theory direction (`D_w > 0`) is descriptive
  only.

## 3. Design (frozen; full detail in the protocol)

One exact pre-history snapshot per world → exact clones → two frozen two-stage histories
(`H_OVER` overshoot-then-starve, `H_APPR` gentle approach) → frozen 1000-step deep turnover
(`M ≤ 0.25`) → match to a **per-world, outcome-independent** target `m*` (= direct-relaxation reference
`Z` at fixed `t_ref`) via a frozen crossing rule (release at first step ≥ `A_min` where `Z` crosses
`m*`, accept if within `tol = 0.02·m0`) → **common release** (same engine, one common
history-independent boundary source on the core/collar, `up_ref = 0` clamped equal, probe-free) →
record `Z(t)` over `T_hor`.

**Frozen scalars (proposed; locked verbatim at seal):** `tol = 0.02·m0`; `A_min`, `t_ref`, `T_hor`
(proposed 120) fixed by mechanical rule, not from any excursion; deep interval 1000; `M ≤ 0.25`.

## 4. Controls (frozen)

Direct-relaxation reference; exact own-replay sham (bit-exact determinism); identical-history clone
control (`D ≡ 0`); one-step discontinuity diagnostic; survival/tracker controls (censor, never impute);
clock-time / branch-age control (record ages; run an equal-age variant); tracker-independent robustness
(frozen-mask primary vs tracked body mass).

## 5. Outcomes (frozen classification)

`HIDDEN_STATE_SUPPORTED` / `Z_SUFFICIENT_NOT_ESTABLISHED` / `MATCH_INVALID` / `UNRESOLVED`, with the
exact Boolean conditions in `KOVACS_HIDDEN_STATE_00_ESTIMAND_AND_CLASSIFICATION.md`. A null is a bounded
`Z_SUFFICIENT_NOT_ESTABLISHED` and is never converted into a proof that `Z` is sufficient or that no
hidden state or memory exists. A divergence with a failed validity block is `MATCH_INVALID`, never
support.

## 6. Forbidden (pre-committed)

No body-equalisation surgery (the match is by history/timing only); no regression residualisation
presented as causal control; no decoder or reader battery; no outcome-selected `Z`; no tuning of `m*`,
`tol`, `A_min`, `t_ref`, `T_hor`, gate, or horizon from any response; no new seed namespace in Phase 0;
no prospective execution in Phase 0; no ownership/identity/reconstruction/heredity/agency claim. The
58xxx DOWNSTREAM-ORDER-READER family is not reopened; V5, 03G, and DIRECTED-CAUSAL-PAIR artefacts are
not modified.
