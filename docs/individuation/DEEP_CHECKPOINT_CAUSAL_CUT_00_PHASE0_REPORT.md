- hard and boundary-repaired transplants retain their inherited `STOP-TRANSPLANT` disposition.

Inventing a new spatial coefficient mask would be operator development, not evidence that a cut currently exists.
Phase 0 therefore stops before clone creation or future-response access.

## What a qualifying local cut would have to mean

A valid cut must remove one predeclared causal equation edge on an outcome-independent physical support at A and/or
B while leaving the serialized state itself unchanged at intervention. It must:

- use the same mechanism and numerical operations at A and B;
- never use diagnostic particle/tracker identity in physics, detection, or association;
- leave the partner, common environment, boundary, probe, body state, clock, scheduler, and all global coefficients
  identical across exact clones;
- touch only the declared edge, not erase its source state or overwrite neighboring state arrays;
- have a manipulation-equivalent sham and a predeclared float64 disturbance envelope;
- compose at A+B without order dependence, overlap, double application, or global side effects;
- remain confined continuously while the pair stays separated, viable, non-fusing, and independently tracked.

No existing operator satisfies this definition.

## Conditional minimal factorial — identification target only

The following design is frozen only as the question that a future, separately authorized operator would have to
identify. It is **not executable with the current repository**.

Starting from one byte-identical full deep checkpoint, including clock and scheduler state:

| Arm | Cut at A | Cut at B | Meaning |
|---|---:|---:|---|
| `C00` | 0 | 0 | unchanged common continuation |
| `C10` | 1 | 0 | same qualified access edge cut only at A |
| `C01` | 0 | 1 | same qualified access edge cut only at B |
| `C11` | 1 | 1 | order-independent composition of the two local cuts |

Each active arm needs a manipulation-equivalent sham with the same support construction, calls, indexing, and
floating-point operations but without severing the equation edge. All arms receive the unchanged corrected common
probe: common nutrient standardization, 40 settle steps, uniform amplitude `0.25` for 5 steps, and a 40-step
response horizon. No history writer, state transplant, body standardization, partner substitution, or branch-specific
clock is allowed.

“Without writing any new history” is strict here. From checkpoint load through settle, probe, and endpoint, every
clone must use `eta_w=0` and record zero external writer calls and zero exposure-dependent `Mf` increment. The
unchanged engine may passively copy already-present `Mf` onto newly created mass and may apply its common frozen
decay/transport terms; this is transport/evolution of the serialized state, not acquisition of new exposure history.
That passive-copy policy and the decay/transport parameter block are hash-bound identically across all arms.

Let `Y_A^{ab}` and `Y_B^{ab}` be separate raw future responses for recipients A and B in arm `Cab`; tracked uptake
is primary and the frozen checkpoint mask is a control. Raw response values are stored by arm only after all whole-
world mechanics pass. The raw record contains no derived contrast.

For each recipient `r ∈ {A,B}`, retain all conditional cut effects:

```text
Δ[r←A | B uncut] = Y_r^00 - Y_r^10
Δ[r←A | B cut]   = Y_r^01 - Y_r^11
Δ[r←B | A uncut] = Y_r^00 - Y_r^01
Δ[r←B | A cut]   = Y_r^10 - Y_r^11
```

The full non-symmetrized marginal matrix is row = recipient, column = cut location:

```text
                 cut A                                             cut B
response A   D[A←A] = 1/2[(Y_A^00-Y_A^10)+(Y_A^01-Y_A^11)]   D[A←B] = 1/2[(Y_A^00-Y_A^01)+(Y_A^10-Y_A^11)]
response B   D[B←A] = 1/2[(Y_B^00-Y_B^10)+(Y_B^01-Y_B^11)]   D[B←B] = 1/2[(Y_B^00-Y_B^01)+(Y_B^10-Y_B^11)]
```

`D[A←B]` and `D[B←A]` are never averaged or forced equal. Factorial interactions remain recipient-specific:

```text
I_A = Y_A^00 - Y_A^10 - Y_A^01 + Y_A^11
I_B = Y_B^00 - Y_B^10 - Y_B^01 + Y_B^11
```

The original world is the statistical unit; clone arms, recipients, and repeated steps are nested measurements.
Four DEV candidates can establish mechanics only, never a reliable probability or confirmatory effect estimate.

These are future distinguishing patterns, not claims from this audit:

| `H_RECIPROCAL_CROSS` | both crossed cells are non-null, reported separately rather than as one pair score |
| `H_COMMON_MODE` | pair and sentinel co-move with a global diagnostic; any global change caused by the purported local cut is first an operator failure |
| `H_RELATIONAL` | `I_A` and/or `I_B` exceeds the frozen interaction and factorial-sham envelope without fusion, overlap, or common-mode leakage |
| `H_MANIPULATION_ARTIFACT` | apparent response follows active-minus-sham boundary, body, nutrient, tracker, or confinement disturbance |
| `H_0` | all four matrix cells and both interactions lie within predeclared equivalence regions under valid controls |

A crossed effect is directed access, not by itself relational individuality. A diagonal effect is local access, not
ownership. An interaction is relational dependence only after additive manipulation and common-mode explanations
are excluded.
