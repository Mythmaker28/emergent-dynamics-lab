## Blocking fact

The legacy worlds assign continuous `(a1,a2)` amplitudes, not categorical `H_A/H_B`. The balanced Latin-square
allocation stated in the historical preregistration is not implemented. An A/B factorial cannot be sealed by
thresholding, ranking, or selecting those realized histories after assignment. No primary estimand below is
currently identified.

## Required design before this draft can be unblocked

- `H_A` and `H_B` must be specified before execution as actual treatment sequences, with a protocol-predicted memory
  direction and no outcome-selected amplitude, threshold, fingerprint, or target pair.
- Assignment must use an independent seed namespace, occur after geometry-only eligibility, include both conditions
  in every original world, use every eligible target, and implement a frozen spatial/target-index blocking schedule.
- Exact target counts by condition and world must be frozen. With `K=3`, any 2:1/1:2 imbalance must be counterbalanced
  by design across worlds; otherwise qualify a design with equal within-world counts. No choice is made here.
- The history intervention must demonstrate, before the probe, a stable post-turnover first stage on complete `Mf`,
  `m_plus`, `m_minus`, spatial organization, body/geometry, nutrient/physical core, and the previously frozen 11-D
  fingerprint. No new feature may be chosen from confirmation feeding.
- Because the legacy nutrient treatment changed body/geometry and `rho/U/V/c/N` in 24/24 assigned DEV targets, a
  valid design must predeclare how `H_0` body-state/readout confounding is excluded without modifying the core state
  whose sufficiency is being tested.

## Intended factorial, not executable yet

- Factor H: assigned `H_A` versus assigned `H_B`.
- Factor K: ordinary coupling versus qualified two-cell standardized boundary with history-independent global input.
- Preserve complete core `Mf`, body, geometry, all local physical fields, and Markov phase.
- Clone only after deep-turnover validity is established outcome-independently.
- Apply the clamp only to the isolated continuation and exactly the same boundary/global rule to both histories.
- Normal frozen engine is primary. `lam_plus=0` is secondary mediation; collapse is interpreted as pathway mediation,
  not absence of storage.

At the original-world level, after averaging targets within assigned condition:

- `delta_coupled = mean_world Y(H_A,K_COUPLED) - mean_world Y(H_B,K_COUPLED)`;
- `delta_isolated = mean_world Y(H_A,K_ISOLATED) - mean_world Y(H_B,K_ISOLATED)`;
- `transport_difference = delta_isolated - delta_coupled`;
- `retention_ratio = delta_isolated/delta_coupled`, undefined when the coupled denominator is inside a frozen
  near-zero region.

`Y` would be the preregistered causal future-feeding contrast at fixed horizon 40 under the common probe. Targets are
never statistical units. Scaling, model fitting, and feature processing occur inside original-world held-out folds.

## Prospective outcome labels — drafted, not sealed

- `LOCAL_HISTORY_TRANSPORT_SUPPORTED`: assigned-history first stage passes; same-direction practically non-negligible
  `delta_isolated`; reference robust; valid manipulation; standardized global input.
- `LOCAL_HISTORY_TRANSPORT_NOT_ESTABLISHED`: isolated contrast is negligible, inconsistent, or too uncertain. Never
  reinterpret as absence.
- `REFERENCE_DEPENDENT`: sign reverses across valid boundary references.
- `GLOBAL_DEPENDENCE`: effect requires unstandardized history-bearing global input.
- `MANIPULATION_INVALID`: clamp, probe, viability, tracking, balance, or disturbance gate fails.
- `FEASIBILITY_FAIL`: frozen minimum valid original worlds is not reached.
- `UNRESOLVED`: patterns do not distinguish the above.

No practical margin, equivalence margin, uncertainty rule, valid-world minimum, seed family, or power target is set in
this blocked draft. Those cannot be estimated until a genuinely assigned contrast has a valid DEV first stage.
