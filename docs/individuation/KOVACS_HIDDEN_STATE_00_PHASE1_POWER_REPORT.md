## 1. Dominant result — a STRONG prospective is infeasible on attrition alone

The valid-world yield for a STRONG prospective family is
`eligible × viable × coincidence-qualified`:

| stage | rate | source |
|---|---:|---|
| eligible (focal target exists) | ~24/24 | parent DEV family 57001–57024 |
| viable (complete deep block) | 17/24 = 0.708 | complete-block worlds actually run |
| **full-overt-panel coincidence-qualified** | **0/17 = 0.000** | Phase-1 coincidence analysis |

`24 × 0.708 × 0.000 = 0` coincidence-qualified worlds. Because the coincidence-qualification rate is
**zero** at a mechanically-defensible tolerance, **no family size can yield enough qualified worlds** —
the STRONG prospective is infeasible independent of any effect size. This is the operative power
conclusion and it aligns with the `STOP_PROSPECTIVE` verdict.

## 2. Generic operating characteristics (for reference only)

If a full-panel coincidence design *did* exist, the frozen original-world gate (≥75 % common sign AND a
95 % two-sided Student-t interval excluding 0) would require, for 80 % power at synthetic standardized
effect `d = mean/SD` on a future estimand:

| d | n (t-interval gate) | per-world P(sign) = Φ(d) | n (≥75 % sign gate) |
|---:|---:|---:|---:|
| 0.3 | 88 | 0.618 | ∞ (Φ<0.75, unreachable) |
| 0.5 | 32 | 0.691 | ∞ (unreachable) |
| 0.8 | 13 | 0.788 | 82 |
| 1.0 | 8 | 0.841 | 12 |
| 1.5 | 4 | 0.933 | 2 |

The **≥75 % sign gate is the binding constraint**: it is unreachable unless the true per-world effect is
large (`d ≳ 0.8`, `Φ(d) > 0.75`), and only becomes practical at `d ≳ 1.0` (n ≈ 12). A DEV family of 24
would be adequate only for a very large, highly-consistent effect — and only if coincidence-qualification
were high, which it is not (§1).

## 3. Family cap and inference rules (would-be, frozen)

Any future prospective family (not authorised) would use: a hard family cap set before execution, **no
adaptive extension**, original-source-world inference only (branches/clones/crossings/time points never
increase `n`), and one run + one independent raw reproduction. None of this is exercised here: Phase 1
recommends STOP, so no family, cap, or namespace is defined or reserved.

## 4. Conclusion

The binding facts are (i) coincidence-qualification rate 0/17 and (ii) a sign gate that needs a large,
consistent effect. Both point the same way: **a STRONG prospective Kovacs family is not warranted.**
