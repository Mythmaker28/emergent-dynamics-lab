## Candidate prospective claim

If all frozen DEV gates support `DOSE_AND_ORDER`, a future independently seeded experiment could test whether the
same counterfactual simulator target core retains and causally expresses both cumulative amount and temporal
structure after deep turnover under standardized inputs. This would be evidence for richer causal experience memory
inside the simulator, not identity, unique local ownership, active reconstruction, reproduction or heredity.

If DEV supports only `DOSE_ONLY`, that result is restricted to low-dimensional cumulative-dose information. If it
returns `ORDER_STATE_WITHOUT_FEEDING`, `NO_TRANSPORT`, `NO_MEMORY_FIRST_STAGE`, `SURVIVAL_EFFECT`,
`MANIPULATION_INVALID`, `UNRESOLVED` or `DEV-FEASIBILITY-FAIL`, this draft remains blocked unless a new pre-data
human-reviewed protocol is written.
