# From Material Turnover to Causal Memory: A Falsification Ladder for Organizational Continuity in a Simulated Droplet

**Title candidates**
1. From Material Turnover to Causal Memory: A Falsification Ladder for Organizational Continuity in a Simulated Droplet
2. A Global Cumulative Memory Survives Turnover in a Continuously-Tracked Droplet; A Second Coordinate Does Not
3. Decodability Is Not Individual Memory: Prospective Causal, Turnover, and Tracking Tests in a Simulated Droplet

## Abstract
We ask whether a persistent simulated droplet can carry an internal memory of its experience that is causal,
transplantable, and robust to replacement of its constituent material — and whether such memory individuates
droplets. Using a reaction-transport "scaffold" droplet with a designed two-component intensive memory field
coupled to nutrient uptake (m+) and attractant production (m-), we run a preregistered ladder of interventions
with grouped, prospective, kill-switched evaluation and continuity-safe entity tracking. Central result:
**under a globally imposed nutrient history, multiple persistent droplets locally encode a similar engineered
cumulative-experience coordinate (h1); a continuously tracked droplet retains this internal causal state during
substantial constituent turnover** (longitudinal holdout on untouched seeds: h1 R² = 0.98, 95% CI [0.97, 1.00]
at old-material fraction M≈0.19, 100% track survival). h1 is causally expressed through uptake and
transplantable into a fresh standardized body (R²=0.61), and exceeds body-size/mass baselines after turnover.
Crucially, h1 is GLOBAL in informational content: it is decodable from essentially any large droplet in the
world, so it does NOT individuate droplets. A second candidate coordinate (h2, temporal-order / value-dispersion)
is decodable at rest but fails to qualify: causal response ~1-D, and deep-turnover held-out R²=0.34 (< 0.50)
under continuity-safe tracking. Methodologically we show that high internal variation, decodability, causal
expression at rest, and largest-component visual continuity each fail to imply a turnover-surviving,
individuating causal memory; and that a largest-component reselection tracker can create spurious continuity
that a longitudinal tracker removes. We claim no life, agency, identity, individuation, reproduction or heredity.

## 1. Introduction
Biological individuals persist while their material turns over. We test a deflationary version: can an internal
state encode history causally, separably from body/environment (transplant), and survive turnover — and does it
make one droplet distinguishable from another (individuation)? Decodability is insufficient (a readout may
exploit residual environment, body size, position, or the mere fact that a globally imposed history is written
into every droplet), and visual "the same blob persisted" is insufficient (tracking must be longitudinal, not
size-reselected). Individuation requires strictly more than persistence and is NOT claimed.

## 2. System and methods
**Droplet / memory / turnover / interventions:** as in the frozen C1c architecture (periodic 64×64 reaction-
transport scaffold; bounded intensive memory m=(m1,m2) written by a local experience signal on two timescales,
templated+diffused, inherited by new mass; m+→uptake, m-→attractant; passive pulse-chase turnover; erase /
common-body transplant / channel ablation / clone ceiling).
**Coordinates:** two-phase GLOBAL nutrient histories; h1=a_early+a_late (cumulative), h2=a_late-a_early (order).
**Entity tracking (TCA-01 correction):** the substrate self-organises into many small droplets. Analyses select
the largest connected component. We show this is RESELECTED per checkpoint (not longitudinal) and can teleport
between blobs; we therefore add a frozen LONGITUDINAL tracker (periodic max mask-overlap; censors on loss,
never uses h1/h2/M/labels) and re-certify all turnover claims longitudinally on untouched seeds. Detection is
periodic-safe (verified translation-invariant).
**Statistics:** grouped leave-history-out ridge; no row-LOO; dev vs prospective separated; sealed hashed
families executed once; donor-level bootstrap CIs; shuffled/constant/trivial-feature nulls.

## 3. Results
(1) Persistent organization under turnover (HMC/HSI); hidden state is a generic causal attractor, not
individuating. (2) Engineered causal h1 memory (IOM/MCM): causal, erasable, transplantable. (3) Temporal order
made readable via m- (70×/73× ablation collapse). (4) Apparent dimensionality corrected by grouped evaluation
(row-LOO leakage 0.57→0.19; rank ~1). (5) Two-coordinate storage at rest after de-saturation (h1,h2 ~0.94).
(6) Causal-response collapse: h1 causal (0.61/0.50), h2 not (−0.04/0.30). (7) Family-dependent h2 turnover;
loss not smoothing/dilution. (8) h2 deep-turnover kill-switch: sealed held-out h2 ≈ 0 at M≈0.21.
(9) **Tracker-continuity audit (TCA-01):** the largest-component tracker reselects (5 switches on the incident
seed) and produced the visible teleport + h2 discontinuity; a frozen longitudinal tracker on UNTOUCHED seeds
gives 100% survival and h1=0.98 (CI [0.97,1.00]) at M≈0.19, with h2=0.34 (< 0.50) unchanged. h1 is
tracker-independent because it is a GLOBAL coordinate. (10) Surviving result: h1 is a one-dimensional, global,
causal, transplantable memory of cumulative experience, retained by a continuously tracked droplet through
turnover; it does not individuate droplets.

## 4. Discussion
The ladder separates memory from decodability; storage from causal expression; passive copying from
reconstruction; material replacement from organizational inheritance; and visual/size-reselected continuity
from longitudinal continuity. h1 storage is LOCAL (in the memory field) and causal expression is LOCAL, but its
informational content is GLOBAL, so it does not individuate. Individuation would require multiple independent,
causal, transplantable, turnover-surviving coordinates AND distinct histories carried by different droplets in
one world — none demonstrated. Limits are of the tested C1c architecture, not the substrate in general.

## 5. Limitations
Engineered (not emergent) memory; simulator-specific dynamics; small entities (~30-55 cells); designed
two-phase GLOBAL histories; low-dimensional cumulative h1 that is global (non-individuating); explicit local
memory copying during growth; the historical analyses used largest-component reselection (corrected here);
mean-field transplant readout; no reproduction; no active reconstruction; no individuation; substrate-general
capacity untested.

## 6. Conclusion
Under a globally imposed history, persistent droplets locally encode a similar engineered one-dimensional
cumulative causal memory; a continuously tracked droplet retains this internal causal state through substantial
material turnover. It does not individuate droplets, and a second decodable coordinate did not qualify as an
independent, causal, turnover-surviving dimension. We report these boundaries as the result. Individuation,
reproduction, and a genome are not warranted by the evidence.
