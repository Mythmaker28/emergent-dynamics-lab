# Future-Architecture Specification (NOT implemented; discussion only)

The current C1c architecture supports one causal turnover-surviving coordinate. A qualified SECOND dimension
would require changes we deliberately did not make. Candidate directions, in increasing departure:
1. **Active local reconstruction** — a bounded local rule under which new material is driven toward a
   history-characteristic distribution rather than passively copying the local value; must be ablatable, use
   no labels, and be tested against passive-copy and wrong-history controls.
2. **Stable multistate / epigenetic-like memory** — a bounded local bistable inheritance so dispersion is not
   erased by linear averaging; tested for erasability and mutation rate, not called DNA.
3. **Compartmentalized / reaction-diffusion maintenance** — internal sub-domains that protect distinct memory
   states from cross-homogenization.
4. **A different dynamical substrate** — only if the reaction-transport substrate is shown insufficient, which
   this work does NOT establish.
No direction is recommended; substrate-general capacity is NOT established. Any future work must not reopen h2
on the current architecture and must preregister a kill-switch.
