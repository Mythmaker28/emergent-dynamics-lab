# Response to Reviewers (V1 -> V2)

We thank the reviewers. Their concerns largely reinforce the paper's own deflationary stance; we have made the
hedges more prominent and added the requested clarifications. No new experiments were run (none are needed for the
concerns raised). Point-by-point:

## Reviewer 1 (ALife)
- **Novelty / engineered memory (R1.1).** Comment: mechanism is engineered; do not over-claim. Response: agreed; we
  added an explicit sentence in the Discussion ("We are explicit about novelty...") stating the mechanism is
  engineered and the contribution is the evaluation methodology. Change: Discussion, novelty paragraph. Unresolved: none.
- **Individuality in title/abstract (R1.2).** Response: the abstract already states $h_1$ is global and
  non-individuating; we added "and clearly delimits the stronger claims" and, in V2, an explicit clause in the
  first abstract sentence. Change: Abstract; title subtitle retained ("...Organizational Continuity...", not "identity").
- **Substrate dependence (R1.3).** Response: added a generalization paragraph (Discussion) stating what we expect to
  transfer (the qualitative pattern and the corrections) and what will not (individuation). Change: Discussion.
- **Conceptual inflation (R1.4).** Response: memory/causal/organizational/individuation are defined operationally in
  the Introduction (four criteria) and in Table (interpretation boundaries, new in V2). Change: Table 10.
- **Six-hypothesis list buried (minor).** Response: promoted to its own Introduction paragraph (already present).

## Reviewer 2 (statistics)
- **Tracker post-selection / $\theta$ (R2.1).** Response: we state that the tracker and $\theta=0.1$ were frozen
  before the hold-out (Methods, tracker spec) and add a sentence (V2) noting the certification is insensitive to
  $\theta\in[0.05,0.2]$ (the tracked entity's overlap is $\gg\theta$ except at genuine loss). Change: Methods.
- **Multiple comparisons (R2.2).** Response: added to the new "Threats to validity" section that the ladder is a
  fixed preregistered sequence with a single declared primary metric per rung, not a search. Change: Threats section.
- **Small $n$ at depth (R2.3).** Response: figure/table captions and Results now state the CI is over 12 donor
  groups (not 36 rows). Change: Results, Table 6 caption.
- **Marginal rank gate (R2.4).** Response: we state the $0.30$ threshold was preregistered and report $0.283$ as a
  strict miss without reinterpretation. Change: Results (already), reinforced in V2.

## Reviewer 3 (protocell)
- **Biological relevance (R3.1).** Response: downgraded to "inspired by / caricature"; added an explicit caveat in
  Limitations that no quantitative correspondence to a physical protocell is claimed. Change: Limitations (present),
  Related work (softened "caricature").
- **Passive copying vs reconstruction (R3.2).** Response: we now consistently say "passive local propagation" and
  flag every use; a dedicated interpretation-boundary table row (Table 10) states new-material acquisition does not
  license active reconstruction. Change: Table 10, Results.
- **Missing chemistry / bistability (R3.3).** Response: strengthened the Discussion to frame the $h_2$ failure as
  the *expected* consequence of linear averaging vs bistable/hysteretic maintenance, with citations. Change: Discussion.
- **Turnover realism (R3.4).** Response: added a sentence that turnover is a passive-cohort bookkeeping construct,
  not chemical exchange. Change: Methods.
- **Define "viable" (minor).** Response: defined (localized single connected entity above a size threshold). Change: Methods/Limitations.

Unresolved limitations retained honestly: engineered (not emergent) memory; simulator specificity; small entities;
global (non-individuating) $h_1$; mean-field transplant; no reproduction; marginal rank gate; tracker developed with
knowledge of one incident seed (mitigated by disjoint-seed certification).
