# Low-probability Ideas

## Hardware shadow

**Outside the critical path.**

Hypothesis: simulated organization could alter the physical execution profile only because spatial structure changes neighbor distributions, bin occupancy, branches, memory access, or effective work. The causal chain would be:

`simulated organization -> backend workload -> CPU/GPU activity -> power or side-channel emissions`.

This is not an electromagnetic signature of simulated particles.

A future controlled test would compare a structured world with a spatially shuffled matched control while holding particle/type counts, law, steps, backend, and rendering fixed; randomize run order and compare a data-dependent backend with a constant-work backend. Disappearance under constant work would favor workload leakage.

Do not pursue before EXP02/EXP03 protocol work.
