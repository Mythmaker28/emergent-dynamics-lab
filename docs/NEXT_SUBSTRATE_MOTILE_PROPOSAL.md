# Next substrate proposal — motile open-system, with the scrambled-cargo null built into the gate

## Why Gray-Scott died
Not because it lacked openness, turnover, or persistence — it had all three. It died because **its structures are
reconstructible from unstructured matter**: drop the right amount of V-rich reagent anywhere receptive and a spot
regrows. A substrate in which the organization of the cargo is causally irrelevant cannot exhibit individuality,
because there is nothing for an individual *to be*.

## The requirement this implies (the real lesson of EXP-RD-04)
A substrate can only support "persistent dynamical individuality" if **the internal organization of a structure is
load-bearing** — i.e. if a scrambled cargo with identical mass and composition **fails** where the intact one
succeeds. This is now the **primary qualification gate**, ahead of any blind law map:

**GATE-0 (scrambled-cargo separability).** On a hand-built reference structure known to be organized, the intact
cargo must re-establish and the scrambled cargo (identical mass, identical composition, identical destination,
identical displacement) must **not**. If a substrate cannot separate these two, it is rejected **before** any law
search — no exceptions, no rescue mechanisms.

Gray-Scott, tested retrospectively, would have **failed GATE-0**. So would (almost certainly) the closed Flow-Lenia
core. This gate would have saved the project two substrates.

## Candidate substrate class: motile, self-propelled, internally structured
The needed feature is **an internal degree of freedom that cannot be reconstituted from a well-mixed lump** —
typically a **polarity / orientation field** that must be spatially coherent to function. Candidates, in order of
preference:

1. **Active-matter / self-propelled particle field with a polarity field** (Toner–Tu-like, open: energy input +
   dissipation). A cluster's coherent polarity is destroyed by scrambling and cannot be regenerated from the same
   material without re-organizing — exactly the load-bearing organization Gray-Scott lacks.
2. **Chemotactic aggregate (Keller–Segel-like) with an internal secretion/response asymmetry.**
3. **Multi-species open RD with an autocatalytic loop whose stoichiometric arrangement is spatially ordered.**

## Non-negotiables carried forward
Passive tracers never influence dynamics. Exact controlled limit. Spatially homogeneous, detector-independent
forcing (the homogeneity null must give ptp = 0). P and M kept separate, thresholds frozen, no composites. Blind
low-discrepancy law map, preregistered before screening. CONTROL / SHAM / PERTURBED / PLACEBO **plus the mandatory
NULL-SC scrambled-cargo arm**. Continued-turnover requirement. Observer-only sensitivity at fixed physics and fixed
`t*`. Explicit denominators, Wilson intervals, prospective replication on unseen seeds.
