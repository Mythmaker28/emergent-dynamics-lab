## The risk that would fool me, named in advance

small manifold genuinely different entities **collapse into one fingerprint** — a collapse that would present itself
as *"identity persists"*, which is the answer this programme has an interest in seeing.

This is not hypothetical. It is **measured**: in development, 4/4 pairs that are DIFFERENT under rich access are
INDISTINGUISHABLE under the droplet repertoire — including a system with a third cause, and a gated state machine.
**In a droplet, entities differing only in an internally-gated cause will collide, and the fingerprint will say so
by calling them an equivalence class, not the same individual.** Every verdict must carry its coverage, and the
word SAME does not exist in this instrument's vocabulary.
