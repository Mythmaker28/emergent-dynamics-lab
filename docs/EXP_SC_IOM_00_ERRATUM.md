## Correction 1 — verdict wording
The IOM-00 causal response decoded continuous NET exposure at held-out R2 ~ 0.95-0.97. The precise
interpretation of `PASS — HISTORY-CLASS MEMORY ONLY` is therefore:

    PASS — LOW-DIMENSIONAL CAUSAL EXPERIENCE MEMORY
    INDIVIDUATION: FAIL

The memory is causal, erasable, transplantable and turnover-surviving, but its CAUSAL read-out collapsed
onto ~one continuous net-dose coordinate.

## Correction 2 — storage claim
Replace "high-dimensional storage" with:
"high-dimensional internal VARIATION (memory-field participation rank ~11); reproducible high-dimensional
history information NOT established." Held-out decoding (IOM-00 dev -> prospective) of the 4 continuous
sector amplitudes from the memory field gives R2 ~ -1.6 (worse than the mean); from the global memory mean
R2 ~ 0.009. The spatial/continuous history is NOT reproducibly stored.

## What IS reproducibly stored (established here)
Temporal ORDER is reproducibly stored in the global contrast m- = m1 - m2: for matched-dose H1 (N->c) vs
H2 (c->N), m-(H1) ~ -0.23 vs m-(H2) ~ -0.34, paired difference 0.116 with std 0.004-0.007 -> 100%
classification on BOTH development and the held-out prospective family. IOM-00's failure to express order
was a READ-OUT limitation (scalar coupling read m+ = m1+m2 only), not a storage failure. This motivates
EXP-SC-MULTI-CHANNEL-ORGANIZATIONAL-MEMORY-00 (freeze writing; add an orthogonal m- read-out channel).
