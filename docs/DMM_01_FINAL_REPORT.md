but SMC-01's turnover **collapse is corrected as a non-robust, underpowered artefact**, and the loss is shown
to be **not a correctable maintenance mechanism**. This is a negative-of-a-negative plus a power limit.

## OBSERVED
- Ablation branches A0/A3/A4/A34 give identical h2 trajectories; memory dispersion grows during turnover.
- Old vs new constituents decode h2 equally (0.77/0.77) at M≈0.44 → new material inherits the distribution.
- Pooled (4-seed) h2: M1.00→0.97, M0.58→0.87 (CI [0.57,0.97]), M0.36→0.67 (CI [−0.21,0.89]).
- Sealed prospective (3-seed, n=12): M1.00→0.87 (CI [0.41,0.97]), M0.60→0.43, M0.37→0.48 (CIs span zero).

## INFERRED
- The frozen C1c memory PASSIVELY carries h2 into new material (local inheritance + uniform death), so h2 is
  not material-bound. But h2 is a low-amplitude value-dispersion coordinate; ongoing growth injects competing
  dispersion, so its held-out decodability through turnover is marginal and high-variance at achievable n.
- SMC-01's confident "does not survive turnover" was an artefact of underpowered, single-family estimation —
  exactly the confound its own erratum (this mission) flagged.

## FALSIFIED
- "h2 is material-bound / turnover erases h2" (SMC-01) — not robust; corrected.
- Homogenization (templating/diffusion) as the h2-loss mechanism — falsified (ablations null; dispersion grows).
- Dilution by zero-memory material — falsified (new material inherits h2 equally).
- My own Phase-A prior that smoothing dominated — falsified.

## SPECULATIVE
- With much larger donor/seed counts and deeper turnover, h2 retention might cross 0.50 robustly — this is a
  question of statistical power, not a missing mechanism. Only then would active-reconstruction be worth testing.

## Independent judgment — should the project keep pursuing multi-dimensional organizational memory?
My honest assessment hardens: the project should **stop escalating and consolidate the one robust result** — a
one-dimensional (cumulative) causal, transplantable, turnover-surviving organizational memory (h1). The second
coordinate h2 has now been chased through four experiments (writing, causal readout, spatial/frame, maintenance)
and at every stage it is real-but-marginal: decodable at rest, frame-free, passively inherited, yet never
robustly causal, never robustly turnover-surviving at the preregistered bar, and always underpowered at the
n the substrate/compute allows. That pattern — a signal that survives each new test only at the noise margin —
is itself the finding: **h2 is a weak epiphenomenal statistic, not a second organizational memory dimension.**
The scientifically correct next step is NOT another mechanism (Phase B) or reproduction/genome, but a single
adequately-powered certification of h2 turnover-retention (≥8 seeds, M≤0.25, grouped CIs). If it fails to clear
0.50, the 1-D conclusion should be written up as the result and the individuation/lineage program closed.
