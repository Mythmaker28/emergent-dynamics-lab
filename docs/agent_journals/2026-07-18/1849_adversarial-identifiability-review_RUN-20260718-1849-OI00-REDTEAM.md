## OBSERVED

- Exact cloning copies all eight persistent arrays plus the step/clock; it creates no causal contrast on its own.
- The common standardized probe resets and pulses `N`; it is a common environmental excitation, not an ownership cut.
- `lam_plus=0` is one world-global coefficient cut on the pointwise `m_plus -> uptake` term. It cannot produce A-only
  and B-only arms.
- `up_ref_zero` changes the global uptake reference in the memory-writer equation; `copy_disabled` deletes passive
  memory inheritance into newly fed material. Neither independently cuts access to an already-formed state.
- Local `Mf` erasure changes the proposed carrier itself. It is local at surgery but need not stay dynamically local,
  and it does not preserve the candidate state.
- Own replay is a qualified no-op only on its recorded unprobed trajectory. Reference replay and the no-swap clamp
  overwrite all eight arrays on a fixed collar after each step. The current engine has one boundary driver, not two
  independently composable A/B drivers.
- Transplant exchanges all eight arrays inside the fixed core and therefore changes carrier, body, local physical
  environment, and the interface at once. Writer interventions add spatial nutrient drive and jointly change memory,
  environment, body, and geometry over time.
- Projected onto desired A/B local access-cut coordinates, the admissible operations provide only `00` (no cut) and
  at most `11` (global readout cut). The projected design has rank at most one and omits `10/01`.

## INFERRED

- Intervention qualification is part of identifiability. A syntactically executable but mechanically rejected map
  cannot be counted as a separating intervention.
- The current ownership-model-to-observable map is non-injective. In ideal clean local/environment access coordinates
  `(l,e)`, the truth tables for local-only, environment-only, redundant-OR, and synergistic-AND models differ only in
  the unobserved single-factor cells `10/01`; all give an active response at `11` and no response at `00`.
- A concrete reachable-manifold construction makes the same point. Let binary history `Z` satisfy `L=E=Z` on all
  uncut reachable states. `H_L: Y=qL` and `H_S: Y=qLE` are identical for cloning/common probing, local erasure
  (`L:=0`), and global readout cut (`q:=0`). An environmental model can match the same observations through an
  environmental address plus a content-free local actuator disabled by destructive erasure. Thus erasure necessity
  does not locate ownership.
- Replays, transplant, and writers do not rescue separation: each opens an intervention-to-disturbance-to-response
  path while changing more than one hypothesized ownership parent. Without exclusion restrictions, that nuisance
  path can reproduce either model's response table.
- The accepted 03G causal result constrains a strict no-causal-history null, but it does not reject an ownership-null
  model in which a persistent state affects feeding without a uniquely addressable owner.
- Final report notation explicitly separates availability surfaces `Y^a` (`1=available`) from member-cut responses
  `Y_i^k` (`1=cut`), and the companion JSON records the same semantics.
- The final intervention table includes all audited writer-path variants: global passive-copy disable, global
  `eta_w=0`, local reference-`Mf` substitution, and the non-compact target-centered external history writer.
- The four non-null ownership models remain explicitly interventionally equivalent on the clean reachable diagonal;

## HYPOTHESIS

The current algebra is non-separating: multiple ownership decompositions induce identical potential-outcome tables
over the qualified intervention support because none independently toggles local and environmental/relational access
while preserving the complete pre-probe state and the other causal parents.

## WHAT WOULD FALSIFY THIS?

For A/B directional addressability, an existing qualified operator that changes exactly one predeclared local
readout coupling, preserves all other state and environmental parents at intervention, composes independently at
A/B, and has an equivalent sham. For the full ownership taxonomy, this alone is insufficient: falsification requires
a second independently manipulable environmental/relational access factor or a defensible exclusion/modularity
assumption that makes its potential outcomes identifiable.

## Unresolved risks

- A spatially gated `lambda_plus(x)` is the cleanest local edge surgery, but it supports only local readout-edge
  necessity in the modified substrate unless paired with an independently qualified environmental/relational cut.
- A localized probe through existing `N` coupling is an instrument only if it adds no new `Mf`-specific response
  path. A new `Mf -> reporter` coupling manufactures addressability by construction.
- A passive shadow reader is observational and cannot break causal equivalence; a shadow that implements
  counterfactual dynamics is a new architecture whose transport to the real engine must be justified.
- Boundary isolation can test sufficiency conditional on an artificial boundary, but may manufacture autonomy and
  requires exact sham/matched-boundary qualification.
- Any modified substrate must prospectively re-establish formation, persistence, deep turnover survival, and causal
  response. No 03G or inherited STOP result can be grandfathered.
- A spatial coefficient mask that only deletes an existing pointwise edge is a legitimate surgical intervention;
  its limitation is externally imposed address and conditional edge-necessity scope. Adding an `Mf`-specific
  reporter path, by contrast, manufactures the tested causal route.
