## INFERRED

The four valid single-target availability supports must use their own namespace, with `1=available`:
`L0E0`, `L1E0`, `L0E1`, and `L1E1`. Their saturated matrix is full rank. The later member cut experiment must use a
different namespace and polarity (`k_i=1` means cut), different qualification gates, and a later experimental
phase. A/B composability is a mechanical property the local port should support now; it is not permission to
estimate `Y_i^k` in the `Y^a` experiment.

A spatial `lambda_plus(x,t)` mask is defensible only as a deletion instrument on the pre-existing
`m_plus -> uptake` equation edge. It can establish conditional local-edge necessity/sufficiency at the declared
probe, support, and modified architecture. It cannot establish local storage or the full ownership taxonomy.

The smallest retained environmental candidate is not a collar-state replay. It is a single declared control over
an **exhaustive bundle of pre-existing outside-parent reads** crossing a frozen target boundary, plus every
pre-existing world-common input used inside the target. It sample-and-holds post-`t0` parent innovations relative
to hash-bound references without writing a state array. For credibility it must enumerate transport, diffusion,
templating, reservoir/common, and `up_ref` parents; leave the common probe outside the factor; log every raw face
contribution; and prove that no ungated environmental/common path remains. If the bundle cannot be shown
exhaustive, or if qualification requires state replay, a dedicated response port, or a new reporter, the correct
result is `STOP-ARCHITECTURE`.

Because this bundle creates conditional isolation by deleting extant edges, it can be an intervention on the
modified architecture; it does not by itself establish a natural environmental compartment. The strongest claim
is access through the declared edge partition. Any claim of locality/exclusion requires the exhaustive-path proof,
pre-probe state identity, sham instrumentation, viability, and no-feedback qualification.

The integrated candidate uses sample-and-hold rather than literal deletion. That is the less destructive of the
audited E constructions because it is exact at `t0` and preserves a hash-bound baseline input. It is still a fixed
read-level boundary and therefore remains unadmitted. A superficially smaller contribution multiplier is worse:
the nonlinear shared `_face_flux` has no unique external summand, while zeroing an additive Laplacian neighbour
creates a sink. Neither construction identifies total environmental absence. The repaired package says this
explicitly and retains `STOP-ARCHITECTURE` as the binding Stage-A failure.

## WHAT WOULD FALSIFY THIS?

- Any changed state bit, clock, probe, or target support before the first active update.
- Any natural/open-port update not bit-identical to the frozen engine for every qualified synthetic finite state.
- Any environmental/common influence reaching the target through an existing path not governed by the
  environmental port.
- Any closed-port response produced through a new reporter, boundary source, replay, or dedicated response path.
- Non-commuting port configuration, mask overlap ambiguity, outcome-dependent support, tracker/diagnostic-ID
  dependence, or port effects outside declared direct equation terms.
- Failure to recover the four exact availability rows or a rank below 4 after sham/nuisance projection.
