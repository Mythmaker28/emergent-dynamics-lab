## A claim I made and then retracted, in the same session
Mid-analysis I wrote that the support-monotonicity violation (all 5 S0-successes fail at S3) was "incoherent with an
individual" and "decisive". **I retract that.** Enlarging the support is NOT a pure "carry more of the entity"
operation: it also transports a **U-depleted halo into the destination**, which locally starves the re-forming spot.
The test is therefore confounded — a genuine individual could fail it for purely chemical reasons. Using it as a
falsifier would have been the exact same error class as the cadence test I had just been corrected for. It is
reported as exploratory and carries no eliminative weight.
