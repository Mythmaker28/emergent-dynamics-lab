## Part 1 — SCRAMBLED-CARGO NULL (the decisive test of individuality)

(random permutation of the cells within the support, applied identically to U, V, CU, CV — total mass, total
cohort composition and destination are exactly preserved; only the internal spatial organization is destroyed).

- If the scrambled cargo re-establishes the phenotype **as often** as the intact one, then the transported
  *organization* carries nothing: the destination chemistry plus the delivered mass do all the work.
  → **individuality REFUTED; retire the claim.**
- If the intact cargo re-establishes and the scrambled cargo does not, the organization itself is causally
  necessary. → candidate survives.

Frozen criterion: intact minus scrambled AUDITED rate must exceed the placebo margin (0.25) on the same units.

## Part 2 — DESTINATION-CHEMISTRY CONFOUND

- Report the AUDITED rate stratified by destination-U quartile, with Wilson intervals and explicit denominators.

## Part 3 — FALSE-POSITIVE RATE OF THE PIPELINE (the missing denominator)

individual. Run the identical audit end-to-end with the candidate replaced by a **non-entity support**: a randomly
located region of the same cell count, drawn from cells below the detection threshold, displaced by the same DELTA,
scored against the same phenotype.
- If the false-positive rate is statistically indistinguishable from 4 %, **the candidate is withdrawn** and

- If it is ~0 %, the 4 % is signal and the candidate is promoted to Level 5 (causal), pending replication.
