# What to watch for (visual falsifier checklist)

1. TRACKED ENTITY: the substrate has MANY small droplets; only the red-outlined LARGEST connected entity
   (~30-55 cells) is analysed. Confirm the outline stays on one coherent blob, not an arbitrary region.
2. NOT WHOLE-GRID: the tracked entity must stay localized; it must NOT fill the grid (that would be dispersal/failure).
3. TURNOVER IS REAL: the old/new panel should go red(old)->blue(new) as M falls 1.00 -> ~0.21; the body should
   persist while its material is replaced (this is organizational, not material, continuity).
4. h1 PERSISTS: m+ (blue line in the time series; orange field) should remain written as M falls -> engineered
   cumulative causal memory surviving turnover.
5. h2 IS STATISTICAL: std(m-) (red line) is a dispersion statistic; a single video cannot show h2 "disappear".
   Its failure is the population/held-out result (docs/paper Fig 6 / h2cert), NOT visually obvious. Do not over-read it.
6. FIXED COLOR SCALES: color scales are fixed across the run; a fading signal must actually fade (no autoscaling trick).
7. NO ENVIRONMENT LEAK: nutrient N / attractant c are shown; h1 persistence should not be explained by a frozen
   environmental gradient (transplant controls in the paper remove the body/environment).
8. NOT BODY SIZE ALONE: the tracked size is printed; h1 must not be explainable by body size alone (paper Fig 2/H1 audit
   shows memory 0.93 vs size+mass 0.64 after turnover).
9. BOUNDARY/WRAP: the grid is periodic; check the entity is not an artefact of periodic wrap or a boundary seam.
10. HONEST LABEL: the title says "engineered cumulative causal memory -- not identity/life/genome". It is an audit
    instrument, not proof by animation.
